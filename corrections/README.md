# Corrections

Corrections are append-only.

This directory is empty unless a published row has been corrected. An empty corrections directory means no public correction note has been appended.

When a published row is wrong:

1. Do not edit the original row.
2. Append a new row with correction lineage.
3. Write a correction note here.
4. Regenerate affected reports with visible corrected totals.

Do not create placeholder correction notes. Correction notes exist only for actual appended correction rows.
